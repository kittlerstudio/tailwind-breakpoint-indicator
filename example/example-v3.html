Example: Tailwind v3 (and older)

Breakpoints are passed manually from tailwind.config.js – no CSS variables on this page.

How to test v3 behaviour
  • This page does not define --breakpoint-* in CSS.
  • Breakpoints are passed into initBreakpointHelper({ breakpoints: { ... } }).
  • In a real v3 project you’d copy from theme.extend.screens or theme.screens.

Breakpoints used here: base, xs (30rem), sm, md, lg, xl, 2xl, 3xl (120rem). Resize the window to see the active one.
